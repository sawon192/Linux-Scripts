# Linux Scripts

## Description

This repository contains simple Bash scripts for common Linux system administration tasks. These scripts were created for learning Linux commands, shell scripting, and system administration.

## Scripts

### 1. Backup Script
Creates a backup of a selected directory and stores it in a backup folder with the current date and time.

### 2. User Management
Performs basic user management tasks such as creating, deleting, and listing users.

### 3. File Monitor
Monitors a directory and reports when files are created, modified, or deleted.

### 4. System Information
Displays useful system information including CPU, memory, disk usage, uptime, hostname, and current users.

## Requirements

- Linux Operating System
- Bash Shell

## How to Run

Give execution permission:

```bash
chmod +x backup_script.sh
chmod +x user_management.sh
chmod +x file_monitor.sh
chmod +x system_info.sh
```

Run a script:

```bash
./backup_script.sh
```

or

```bash
bash backup_script.sh
```
